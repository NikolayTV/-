Занятие деятельностью по защите информации без лицензии влечет штрафы:

- Для граждан: 500-1000 рублей
- Для должностных лиц: 2000-3000 рублей
- Для юридических лиц: 10000-20000 рублей

Занятие деятельностью по защите государственной тайны без лицензии влечет штрафы:

- Для должностных лиц: 4000-5000 рублей
- Для юридических лиц: 30000-40000 рублей
