За работу по трудоустройству россиян за границей без лицензии или с нарушением ее условий предусмотрены штрафы:

- Для граждан: 2000-4000 рублей
- Для должностных лиц: 35000-40000 рублей
- Для юридических лиц: 400000-500000 рублей

За грубые нарушения условий лицензии штрафы выше:

- Для должностных лиц: 45000-50000 рублей
- Для юридических лиц: 700000-800000 рублей или приостановка деятельности до 90 дней
