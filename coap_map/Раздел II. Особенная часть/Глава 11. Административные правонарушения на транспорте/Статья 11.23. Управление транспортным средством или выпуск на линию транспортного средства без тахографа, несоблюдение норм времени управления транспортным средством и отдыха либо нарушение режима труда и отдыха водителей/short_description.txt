За нарушение правил использования тахографов и режима труда и отдыха водителей предусмотрены штрафы:

* Управление без тахографа или с его нарушением: 3000-5000 рублей для водителя.
* Выпуск на линию без тахографа или с его нарушением: 7000-10000 рублей для должностных лиц, 15000-25000 рублей для ИП, 20000-50000 рублей для юридических лиц.
* Нарушение норм времени управления и отдыха: 1500-2000 рублей для водителя, 7000-10000 рублей для должностных лиц, 15000-25000 рублей для ИП, 20000-50000 рублей для юридических лиц.
