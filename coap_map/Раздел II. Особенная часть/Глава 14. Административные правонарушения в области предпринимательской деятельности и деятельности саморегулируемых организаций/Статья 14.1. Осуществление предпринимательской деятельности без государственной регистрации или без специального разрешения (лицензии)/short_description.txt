**Основные положения:**

Заниматься бизнесом без регистрации или лицензии, а также нарушать условия лицензии является административным правонарушением.

**Штрафы за нарушения:**

* Ведение бизнеса без регистрации: 500-2000 рублей
* Ведение бизнеса без лицензии: 2000-2500 рублей для граждан, 4000-5000 рублей для должностных лиц, 40000-50000 рублей для юридических лиц
* Нарушение условий лицензии: предупреждение или штраф 1500-2000 рублей для граждан, 3000-4000 рублей для должностных лиц, 30000-40000 рублей для юридических лиц
* Грубое нарушение условий лицензии: штраф 4000-8000 рублей для граждан, 5000-10000 рублей для должностных лиц, 100000-200000 рублей для юридических лиц
