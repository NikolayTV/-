За нарушение правил торговли лекарствами предусмотрены штрафы:

* Для граждан: 1500-3000 рублей
* Для должностных лиц: 5000-10000 рублей
* Для юридических лиц: 20000-30000 рублей

Отдельно штрафуется нарушение требований по установлению предельных надбавок на лекарства:

* Для должностных лиц: 250000-500000 рублей
* Для предпринимателей: двукратный размер излишне полученной выручки
* Для юридических лиц: двукратный размер излишне полученной выручки
