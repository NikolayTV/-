За порчу жилых помещений, их оборудования или использование не по назначению предусмотрены штрафы:

* Для граждан: 1000-1500 рублей
* Для должностных лиц: 2000-3000 рублей
* Для юридических лиц: 20000-30000 рублей

За самовольное переустройство или перепланировку помещения в многоквартирном доме штрафы составляют:

* Для граждан: 2000-2500 рублей
* Для должностных лиц: 4000-5000 рублей
* Для юридических лиц: 40000-50000 рублей
